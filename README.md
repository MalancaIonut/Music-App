# Music-App
### This is a music application. 

# Functionalities :
- Top 15 songs.
- Sign up / Login / Logout.
- Admin role :
  - Create a new song.
  - Control all users.
- List of all songs :
  - Search in list for a specific song.
  - Vote your favorite song.
  
# Technologies:
- Express.js
- Postgresql
- Bootstrap
